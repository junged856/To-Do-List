# To-Do-List
A simple to do list app made in JavaFX using Eclipse

<h1>Features:</h1>
<p>- Can select tasks</p>
<p>- Can "complete" checkmarked tasks</p>
<p>- Able to add new tasks</p>
<p>- Able to toggle between selecting and de-selecting all tasks</p>
